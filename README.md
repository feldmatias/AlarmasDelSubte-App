### Setup
Install node.js and npm, then run `npm install`

### Build
- Run tests: `npm test`
- Run linter: `npm run pretest`

### Git hooks
Copy all files in `/hooks` to `.git/hooks/`

They will run the linter for every commit, and tests before a push.
